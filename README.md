#Remedy.Dev is a free, browser-based tool for developers that instantly documents any code project.
What you do: Drop your project folder into the site. That's it.
What you get back:

A full breakdown of your codebase — languages, frameworks, file structure
Auto-discovered API endpoints and routes
Architecture diagram showing how your layers connect
A ready-to-use GitHub README you can copy or download
Setup guide with the exact install and run commands for your stack
Improvement suggestions ranked by priority

Who it's for: Any developer who needs to quickly understand a new codebase, document an existing project, or generate context for sharing with teammates or AI tools.
Why it's useful: No signup, no upload, no backend — everything runs in your browser. Your code never leaves your machine. Works with every major language and framework: JavaScript, Python, Go, Rust, Java, C#, PHP, Ruby, Swift, Dart, and more.
The one-liner: Drop your code in, get your docs out.
